# bean-board 🖼️

The bean-board for bean parties!

## About 💡

The UI to upload images to Daniel's bean-board!

## Usage 🔨

### Python 🐍

To run locally with Python, just do the following:

```bash
python3 app/app.py
```

### Docker 🐳

```bash
make run
```

Then browse to:

- `localhost/button`
- `localhost/basic`
